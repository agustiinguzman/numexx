# Numexx

Sitio web desarrollado como proyecto final del curso de Desarrollo Web en Coderhouse.  
Incluye estructura con HTML5 semántico, SCSS modularizado con partials, uso de Flexbox y diseño responsive inspirado en el estilo minimalista de Apple.

## 📁 Estructura del proyecto

- `/index.html` → Página principal
- `/views/` → Resto de las páginas internas
- `/assets/` → Imágenes
- `/scss/` → Archivos SCSS separados por partials
- `/css/style.css` → Estilo final compilado

## 🚀 Funcionalidades

- Navegación responsive con menú hamburguesa
- Favicon, imágenes con alt, tipografía profesional estilo Apple
- Código organizado, limpio y listo para escalar

## 🧠 Herramientas usadas

- HTML5
- SCSS (con partials, variables y mixins)
- Git + GitHub
- Live Server para pruebas locales
